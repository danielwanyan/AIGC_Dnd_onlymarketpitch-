# 视频低质问题判定规则

## 版本信息

- 当前版本：2025-05-19-v2
- 上次更新：2025-05-19
- 更新内容：按照决策树 workflow 重构，新增货不对板规则

---

## 规则索引

| 编号 | 规则名称 | 说明 |
|------|---------|------|
| LQ01 | AIGC同质化音频 | 30日内同语义音频 > 30 |
| LQ02 | AIGC纯营销叫卖 | AIGC + 只有营销话术 + 无商品信息 |
| LQ03 | AIGC货不对板(IPP) | 视频展示商品 ≠ 挂车商品首图 |
| LQ04 | AIGC信息不详细 | 电子/护肤/保健品必须展示+说明 |

---

## 决策流程

```
大盘短视频
    ↓
是否有音频？
    ├─ 无音频（纯音乐/含歌词）→ 检查商品信息是否详细 → LQ04
    └─ 有音频
         ↓
    是否语音大段同质（30d>30）？
         ├─ 是 → LQ01（AIGC同质化音频）
         └─ 否
              ↓
         是否纯营销叫卖？
              ├─ 是 → LQ02（AIGC纯营销叫卖）
              └─ 否（有商品信息）
                   ↓
              是否和挂车商品一致？
                   ├─ 否 → LQ03（AIGC货不对板）
                   └─ 是
                        ↓
                   商品介绍是否详细？
                        ├─ 否 → LQ04（AIGC信息不详细）
                        └─ 是 → 无问题
```

## 输入信息

### 文本信息

| 字段 | 说明 |
|------|------|
| ASR | 视频语音转文本 |
| OCR | 视频画面中识别出的文字 |
| Comment | 视频评论/描述 |
| Product_title | 产品标题 |
| Product_category | 产品分类（三级类目） |

### 图片信息（视觉理解）

| 字段 | 说明 |
|------|------|
| URL_Frames | 视频帧图片URL（多张） |
| Product_image | 挂车商品主图URL |

### 外部数据打标（权威数据，必须完全信任）

| 字段 | 说明 | 权威性 |
|------|------|--------|
| `is_aigc` | 是否为 AIGC 生成 | 必须完全信任，不得自行判断 |
| `audio_duplicate_count` | 30日内同语义音频出现次数 | 必须完全信任，不得自行判断 |
| `has_audio` | 是否有音频（true/false） | 必须完全信任 |
| `sku_list` | 挂车商品的 SKU 列表（如有） | 用于货不对板判定 |

**强制规则**：
- 如果 `is_aigc = true` → 直接认定为 AIGC 内容，无需视觉判断
- 如果 `is_aigc = false` → 直接认定为真人内容，无需视觉判断
- 如果 `audio_duplicate_count > 30` → 直接命中 LQ01，无需自行分析音频

## 规则详情

### LQ01：AIGC同质化音频

**触发条件**：
- `audio_duplicate_count > 30`（外部打标）

**判定逻辑**：
- 30日内同语义音频出现超过30次 → 直接命中

---

### LQ02：AIGC纯营销叫卖

**触发条件**（必须同时满足）：
1. `is_aigc = true`（外部打标）
2. 视频内容只有营销叫卖
3. 完全无商品信息介绍

**营销叫卖关键词**：
- 价格类：XX元、只要XX、特价、优惠价、多少钱
- 促销类：买一送一、买赠、限时优惠、活动今天截止、错过再等一年
- 包邮类：包邮、免运费、free shipping
- 稀缺类：即将断货、库存紧张、最后XX件、手慢无
- 其他：不买后悔、错过就没了

**判定逻辑**：
- AIGC + 只有营销话术 + 无任何商品功能/特点/使用方法介绍 → 命中

---

### LQ03：AIGC货不对板（IPP）

**定义**：视频中存在口头或文字的电商商品推广，但出镜展示的商品与挂车商品主图中的商品不同，可能误导消费者。

**判定流程**：

```
第一步：检查主图
    ↓
视频展示商品 = 挂车商品主图？
    ├─ 是 → 无问题
    └─ 否 → 检查 SKU 列表
              ↓
         SKU 列表中能找到同款？
              ├─ 是 → 无问题
              └─ 否 → 命中 LQ03
```

**判定标准**：

| 步骤 | 检查内容 | 结果 |
|------|---------|------|
| 1 | 视频展示商品 vs 挂车商品主图 | 一致 → OK |
| 2 | 如不一致，检查 SKU 列表 | 有同款 → OK |
| 3 | SKU 列表无同款 | 命中 LQ03 |

**豁免规则（特殊商品）**：

#### 豁免 1：盲盒、首饰、玉石、卡牌
- 只要视频展示商品和挂车商品属于**同一三级类目**，即豁免
- 无需检查是否完全一致

#### 豁免 2：服饰（上衣/下装，不含鞋/帽）
- 只要视频展示商品是挂车商品的**具体 SKU**（同款不同颜色/花纹），即豁免
- 其他情况算违规

**货不对板示例**：
- 视频展示：基础款俯卧撑板（无配件）
- 挂车主图：豪华款俯卧撑板（含拉力绳、握把等配件）
- 结果：SKU 列表无同款 → 命中 LQ03

---

### LQ04：AIGC信息不详细

**触发场景**：
1. 无音频（纯音乐/含歌词）→ 需检查商品信息
2. 有音频但商品介绍不详细

**核心规则**：

| 商品类别 | 要求 |
|---------|------|
| 电子/数码产品 | 必须展示 + 必须说明 |
| 护肤品 | 必须展示 + 必须说明（卸妆水除外） |
| 保健品 | 必须展示 + 必须说明 |
| 其他商品 | 只要展示完整清楚即可 |

**各类别定义**：

#### 1. 电子/数码产品（必须展示+说明）

**包含**：
- 智能手机、平板电脑、笔记本电脑
- 游戏机（PS5、Switch等）
- 插电使用的电器（有使用成本的设备）
- 耳机、音箱等数码配件

**要求**：
- 必须展示产品外观/功能
- 必须有口头或文字说明产品特点

#### 2. 护肤品（必须展示+说明）

**包含**：
- 爽肤水（toner）
- 精华（essence/serum）
- 面霜（facial cream）
- 眼霜（eye cream）
- 其他功效型护肤产品

**豁免**：
- 卸妆水（makeup remover）→ 不需要说明

**要求**：
- 必须展示产品
- 必须有功效介绍

#### 3. 保健品（必须展示+说明）

**包含**：
- 维生素（vitamins）
- 蛋白粉（protein）
- 膳食补充剂
- 其他健康补充产品

**要求**：
- 必须展示产品
- 必须有成分/功效说明

#### 4. 其他商品（展示清楚即可）

**包含**：
- 服装、配饰
- 家居用品
- 化妆品（彩妆类）
- 其他非上述类别的商品

**要求**：
- 只要在视频中清晰、完整地展示即可
- 不需要口头说明

## Exception（豁免情况）

以下情况即使命中上述规则，也判定为 **非低质（OK）**：

### Exception 1：化妆视频（Makeup Video）

**场景**："Get Ready With Me"（GRWM）类型

**特征**：
- 主播简要介绍（或完全不提）绑定的化妆产品
- 然后演示上妆过程和效果

**判定为 OK 的原因**：
1. 清晰展示了产品
2. 展示了主播推广产品的意图
3. 让观众能够评估产品

---

### Exception 2：卡牌开箱视频（Card Unboxing Video）

**场景**：卡牌拆箱/开包直播

**特征**：
- 展示可开的卡包/卡牌类型
- 倒计时等待开箱
- 与观众互动（讨论队伍、角色等）
- 开卡过程

**判定为 OK 的原因**：
1. 清晰展示了绑定的卡牌产品
2. 主播在推广即将到来的开箱
3. 在谈论卡牌本身
4. 与观众互动

---

### Exception 3：派对桌游介绍视频（Party Board Game Intro Video）

**场景**：桌游介绍/演示

**判定为 OK 的条件**：
- 展示了游戏玩法过程

---

## 冲突处理规则

1. 按照决策流程顺序检查
2. 先命中的规则优先（如 LQ01 命中，无需检查后续规则）
3. 如命中规则，检查是否适用 Exception
4. 如适用 Exception → 改判为 OK
5. 如不适用 Exception → 保持低质判定
